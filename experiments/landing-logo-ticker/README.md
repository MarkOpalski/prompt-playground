# Logo Ticker Component

A smooth, infinite scrolling logo carousel component based on the 3BL.com ticker design.

## Features

- Infinite Smooth Scrolling
- Dual Direction
- Responsive Design
- Hover Effects
- Performance Optimized
- Customizable Speed
- Fallback Support

## Getting Started

Open `index.html` in your browser. Add your logos to the `images/` directory.
